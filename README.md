# sequence-solver
Python scripts for calculating nth terms, partial sums, and recursive sequences. Exploring the intersection of discrete math and algorithmic efficiency.
This project implements core concepts from Precalculus (Sequences, Series, and Finite Differences) into a Python-based analysis tool.

Key Features
Finite Difference Analysis: Detects quadratic growth patterns in datasets.

Term Calculation: Explicit formulas for Arithmetic and Geometric progressions.

Partial Sums: Automated summation for large datasets.
